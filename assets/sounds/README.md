# Notification Sounds

Place your notification sound file here:
- `notification.mp3` - Default notification sound for interval changes

You can use any short MP3 file (< 2 seconds) for workout notifications.

Free sound resources:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

Recommended: Short beep or bell sound for interval transitions.
